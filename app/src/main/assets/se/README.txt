Sound effects assets directory.

Naming:
- se_correct_<deck_id>.wav|mp3|ogg|m4a
- se_incorrect_<deck_id>.wav|mp3|ogg|m4a
- se_correct.<ext>
- se_incorrect.<ext>
- good.<ext> / ok.<ext> (correct alias)
- notgood.<ext> / ng.<ext> / wrong.<ext> (incorrect alias)

Examples:
- se_correct_mtg_story.wav
- se_incorrect_toeic_business.mp3
- se_correct.ogg
- se_incorrect.ogg
